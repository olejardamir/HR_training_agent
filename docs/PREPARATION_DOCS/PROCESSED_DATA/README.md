# Processed Data (Data Preparation Tool)

This directory contains massaged, cleaned, source-attributed, human-reviewable raw data generated by `tools/data_prep/run_data_prep.py`.

## Status

- **Candidate only**: These outputs are not final policy, training content, or runtime data.
- **Review required**: All material must be reviewed by a human before any DB/RAG/runtime use.
- **Runtime approved: false**: No runtime agent, database, fixture, or RAG system uses this data.

## Directories

- `cleaned_sources/` - One Markdown file per readable raw source with source metadata
- `review_packets/` - Topical review files, source evidence index, and decision log template
- `source_attribution.md` - Attribution records tracing every source to its outputs

## Marker

The `.generated_by_data_prep` marker file indicates these files were generated by the data-prep tool and may be safely overwritten with `--force`.
